# DemoControlValueAccesor

Se creó un componente personalizado(StarRatingComponent) que es reconocido como un control de formulario. Este componente implementó la interfaz ControlValueAccessor y agregó el proveedor NG_VALUE_ACCESSOR para ser reconocido como un elemento de formulario.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
